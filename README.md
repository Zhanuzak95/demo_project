# demo_project
---MyBoards - Демо-проект управления проектами
Учебный демо-проект, демонстрирующий работу с массивом объектов и динамическое отображение данных в React-приложении.

---Описание
Проект представляет собой интерфейс для управления списком досок (boards).

---Установка и запуск
1) Клонируйте репозиторий:
git clone <ссылка-на-репозиторий>
cd demo_project
2) Установите зависимости:
npm install
3) Запустите проект в режиме разработки:
npm run dev
4) Откройте в браузере: http://localhost:5173
